# Valentine's Day 2025 - Interactive Page

A playful Valentine's Day interactive web page where users are asked "Do you love me?" with a fun twist - the "No" button moves randomly when you try to hover over it!

## Features

- Beautiful gradient background (pink to peach)
- Interactive "Yes" and "No" buttons
- The "No" button runs away when you try to hover over it
- Clicking "Yes" reveals a romantic message and changes the GIF
- Fully responsive design

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Deployment

This project is ready to be deployed on Vercel. Simply connect your GitHub repository to Vercel and it will automatically deploy.

## How to Use

1. Open `index.html` in a web browser
2. Try to click the "No" button (it will move away!)
3. Click "Yes" to see the romantic message
